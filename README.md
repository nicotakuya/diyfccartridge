# DIY FC Cartridge

## Overview

Famicom homebrew.

## files

FC_CARTV2.COMP : Cartridge's PCB CAD data（for CADLUS X）.

FC_WR.COMP : Flash memory writer's PCB CAD data（for CADLUS X）.

wr.prg : Writing software（for Pi STARTER）.
